Latest Updates

  1. allfunctions.php file use for all functions by use magento class i.e user list, product list. category list etc.
  2. list_users.mag file get all users details which is create our store, means this function provide all users list and allfunctions.php file include in list_users.mag file to call Object.
  3. cat_list.mag file get all categories list which is create our store, means this function provide all categories list and allfunctions.php file include in cat_list.mag file to call Object.
  4. product_list.mag file get all products list which is create our store, means this function provide all products list and allfunctions.php file include in product_list.mag file to call Object.
  5. list_newsletter.mag file get all newsletter list which is create our store, means this function provide all newsletter lists and allfunctions.php file include in list_newsletter.mag file to call Object.

1. User List Show

We need to pass store id in our custom function to get user data.

For Show user list we need to run code in our file

                            
								require_once ("allfunctions.php");
								$users = new Allfunctions();
								$showlist=$users->getUsers(1);
								print_r($showlist->getData());
